# 🔥 Habit Tracker

A web-based Habit Tracker application that helps users build consistency by tracking daily habits and maintaining streaks.

## 🚀 Live Demo
👉 https://pro-habit-tracker-bb9c8.web.app

---

## ✨ Features

- ➕ Add and delete habits
- 🔥 Track daily streaks
- 🏆 Track longest streak
- 📊 Progress visualization
- 💾 Data stored using Firebase Firestore
- 🌐 Fully deployed using Firebase Hosting

---

## 🛠 Tech Stack

- HTML  
- CSS  
- JavaScript  
- Firebase Firestore  
- Firebase Hosting
